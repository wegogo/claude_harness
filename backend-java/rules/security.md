# 安全规范

---

## 一、编码安全

| 类别 | 要求 |
|------|------|
| **SQL 注入** | 使用 MyBatis-Plus 条件构造器或 `#{}` 参数化查询，禁止 `${}` 拼接 |
| **XSS** | 输入过滤 + 输出编码，使用 Spring Security 的 XSS 过滤器 |
| **密码存储** | 使用 BCrypt 加密，禁止明文或 MD5 存储 |
| **敏感数据传输** | HTTPS 加密传输 |
| **日志脱敏** | 日志中不打印密码、身份证、银行卡等敏感信息 |

---

## 二、接口安全

接口层面的安全要求（限流、幂等性、数据脱敏等）参见 [api-conventions.md](api-conventions.md#八接口安全)。

---

## 三、接口鉴权

- 所有接口默认需要鉴权，白名单显式配置
- 不仅要验证身份，还要验证**数据级权限**（用户 A 不能操作用户 B 的数据）

---

## 四、依赖安全

- 定期扫描依赖漏洞（OWASP Dependency Check）
- 禁止使用已知有漏洞的依赖版本
- 关注 CVE 公告，及时升级修复
